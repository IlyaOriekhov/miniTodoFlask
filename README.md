# miniTodoFlask

Простий веб-застосунок для керування списком справ (To-Do List), створений на Flask.

## Опис

Мінімалістичний застосунок для відстеження задач з базовими можливостями створення, редагування та видалення справ. Використовує тимчасове сховище даних у пам'яті (fake_db) для демонстраційних цілей.

## Функціональність

- Перегляд усіх задач
- Додавання нових задач
- Позначення задач як виконаних/невиконаних
- Видалення задач
- Флеш-повідомлення про помилки

## Технології

- Python 3
- Flask 3.1.2
- Jinja2 (шаблонізація)

## Встановлення

1. Клонуйте репозиторій:
```bash
git clone https://github.com/IlyaOriekhov/monoTodoFlask.git
cd monoTodoFlask
```

2. Встановіть залежності:
```bash
pip install -r requirements.txt
```

3. Запустіть застосунок:
```bash
python app.py
```

Застосунок буде доступний за адресою `http://127.0.0.1:5000/`

## Структура проєкту

```
monoTodoFlask/
├── app.py              # Головний файл застосунку
├── requirements.txt    # Залежності проєкту
├── static/            # Статичні файли (CSS, JS)
└── templates/         # HTML шаблони
```
